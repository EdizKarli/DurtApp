# DurtApp
DurtApp

Buraya not yazabilirsiniz.

Program çalıştırma yolu=

1- Terminal aç.
2- Split terminal de ve ikinci bir tane daha aç.
3- İlk terminalde backend dosyasına gel ve "npm start" yaz, Node.js successfull ve Database Handshake successfull mesajlarını alırsan çalışıyor demektir.
4- İkinci terminalde durt_app dosyasına gel. "flutter run" yaz, sana programı nerede açmayı istediğini soracak şimdilik "2" de ve Google Chrome'u seç. Çalışırsa kendisi açacak zaten.

Bilgi = Frontend kodları durt_app/lib içindeki dosyalarda. main.dart asıl çalışan ana merkez.

Telefonda nasıl gözüküyor diye bakmak için = Device olarak Chrome seçin, uygulamayı çalıştırın, sağ tıka basıp İncele deyin, Toggle the Device Toolbar'a basın, açılan sayfanın Responsive kısmına tıklayıp istediğiniz telefonu seçin.
